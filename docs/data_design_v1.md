# 資料設計 v1：子集篩選策略與排除規則

> 對應腳本：`experiments/03_build_corpus.py`
> 本文件記錄「哪些案件被納入、哪些被排除、為什麼」，作為未來可重現性與透明度的依據。

---

## 1. 資料來源

- 司法院開放資料（opendata.judicial.gov.tw）
- 涵蓋：刑事、民事（本專案不含行政、懲戒）
- 原始量級：刑事 532,276 筆 / 民事 1,074,639 筆

---

## 2. 時間篩選

只納入**民國 100 年（西元 2011 年）後**的判決。

理由：
- 近年賠償金額、法律見解具現實參考價值
- 舊案（尤其 89-99 年）賠償數字有通膨落差
- 控制資料集量級

排除量（估算）：刑事約 40 萬筆、民事約 65 萬筆降為舊案。

---

## 3. 第一刀：排除程序性案件

程序性案件**沒有事實描述與法律推理**（如「支付命令」只有一行「命被告給付 XX 元」），對 RAG 類案檢索完全沒有意義，應在最前端排除。

### 3.1 刑事排除規則

| 排除條件 | 具體判斷 | 排除量（刑事全量） |
|---------|---------|-----------------|
| JCASE 含「聲」 | 聲明異議、各種聲請 | ~10,466 筆 |
| 案由含「聲明異議」 | 同上（部分案由以文字標示） | 含於上 |
| 案由為「定應執行刑」 | 純量刑程序，無事實 | 少量 |
| 案由為「宣告沒收」 | 純沒收程序 | 少量 |

**刑事排除邏輯（程式碼）：**
```python
def is_criminal_procedural(jcase: str, title: str) -> bool:
    if "聲" in jcase: return True
    if title in ("定應執行刑", "宣告沒收", "聲明異議"): return True
    return False
```

### 3.2 民事排除規則

依 JCASE 模式與案由關鍵字兩層判斷：

**依 JCASE 排除：**

| JCASE | 意義 | 排除原因 |
|-------|------|---------|
| 司促 | 支付命令(司法促請) | 純催債命令，無法律推理 |
| 司票 | 本票裁定 | 純程序裁定 |
| 除 | 除權判決 | 程序性 |
| 補 | 補充判決 | 程序性 |
| 消債更 | 消費者債務更生 | 破產程序，非爭議判決 |
| 聲/台聲 | 聲請事件 | 程序性聲請 |
| 台抗 | 抗告 | 程序性 |

**依案由排除（含以下關鍵字且非實體爭議）：**

| 案由模式 | 排除原因 |
|---------|---------|
| 支付命令 | 催債命令 |
| 本票裁定 | 票據程序 |
| 除權判決 | 票據程序 |
| 公示催告 / 公示送達 | 程序性公告 |
| 確定訴訟費用額 | 費用計算程序 |
| 更生事件 | 破產程序 |
| 給付信用卡... / 給付電信費 / 給付簽帳卡... | 純金錢催收，無法律爭議 |
| 清償債務 / 清償借款 / 返還借款 / 清償信用卡... | 同上 |

**保留邏輯（反向）：** 只要案由屬於以下類型，即使金額相關也保留：
- `損害賠償`（各種類型）
- `侵權行為`
- `遷讓房屋`、`分割共有物`（有實體爭議）
- `離婚`、`監護`、`扶養`（身分事件，有法律論述）
- `返還不當得利`
- `給付工程款`、`給付貨款`（商業爭議，有事實描述）

---

## 4. 第二刀：分層抽樣（控制量級，保留多樣性）

### 4.1 為什麼要分層？

排除程序性後，民事「損害賠償」一個案由可能仍有 2 萬筆，
而「分割共有物」只有幾百筆。直接隨機抽樣會讓大類完全壓過小類，
導致系統對少見案件類型的檢索能力極差。

### 4.2 分層策略

- **分層依據**：`JTITLE`（案由）
- **每層上限**：**200 筆**
- **邏輯**：若該案由 ≤ 200 筆 → 全部保留；> 200 筆 → 隨機抽 200 筆（seed=42）

### 4.3 效果

```
損害賠償 29,598 筆 → 抽 200
侵權行為損害賠償 14,011 筆 → 抽 200
損害賠償(交通) 3,907 筆 → 抽 200
分割共有物 ~500 筆 → 抽 200
離婚 ~500 筆 → 抽 200
給付工程款 ~300 筆 → 抽 200
買賣 ~100 筆 → 全留
```

每種案由都有代表，不被大類蓋掉。

## 6. 不納入說明

| 類別 | 原因 |
|------|------|
| 行政判決 | 非本專題範圍（行政訴訟法律體系不同） |
| 懲戒法院 | 量少且針對公務員，使用情境不符 |
| 最高法院（部分） | 事實描述少，多為法律見解，RAG 效果差 |
| 民國 99 年前 | 賠償金額通膨差距大，參考價值低 |
